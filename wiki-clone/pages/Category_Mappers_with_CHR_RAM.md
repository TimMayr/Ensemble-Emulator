# Category:Mappers with CHR RAM

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Category%3AMappers_with_CHR_RAM) | View [other pages](Special_AllPages.xhtml#Category_Mappers_with_CHR_RAM)

Apart from several [Nintendo MMC1](MMC1.xhtml "MMC1") games and a few [Nintendo MMC3](MMC3.xhtml "MMC3") games, most games using CHR RAM were based on [discrete logic mappers](Category_Discrete_logic_mappers.xhtml "Category:Discrete logic mappers").
