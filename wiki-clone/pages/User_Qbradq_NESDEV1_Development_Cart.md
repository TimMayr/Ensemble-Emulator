# User:Qbradq/NESDEV1 Development Cart

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/User%3AQbradq/NESDEV1_Development_Cart) | View [other pages](Special_AllPages.xhtml#User_Qbradq_NESDEV1_Development_Cart)

The NESDEV1 Development Cart is a proposal for a yet-to-be designed development cartridge featuring the [Sunsoft FME-7](Sunsoft_FME_7.xhtml "Sunsoft FME-7") mapper (without extra sound) and a USB interface for loading ROM images. 

## Overview

  * Sunsoft FME-7 Mapper
  * USB interface for reading and writing PRG ROM, PRG RAM and CHR ROM
