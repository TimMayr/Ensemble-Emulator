# Category:Mappers using $4020-$5FFF

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Category%3AMappers_using_%244020-%245FFF) | View [other pages](Special_AllPages.xhtml#Category_Mappers_using__4020__5FFF)

These circuit boards and ASIC mappers map something other than open bus to $4020-$5FFF on the CPU bus. Many of them would interfere with the [Vs. System](Vs__System.xhtml "Vs. System")'s coin counter output.
