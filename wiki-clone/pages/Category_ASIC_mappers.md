# Category:ASIC mappers

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Category%3AASIC_mappers) | View [other pages](Special_AllPages.xhtml#Category_ASIC_mappers)

ASIC [mappers](Mapper.xhtml "MMC") are, as the name suggests, [custom-made integrated circuits](https://en.wikipedia.org/wiki/Application-specific_integrated_circuit "wikipedia:Application-specific integrated circuit") used to perform memory mapping tasks that are too complex or cost-prohibitive to implement using discrete components. Many ASIC mappers also include additional features such as interrupt counters, nametable [mirroring](Mirroring.xhtml "Mirroring") controls, and sometimes even additional sound channels.
