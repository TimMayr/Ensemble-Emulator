# Talk:INES Mapper 048

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Talk%3AINES_Mapper_048) | View [other pages](Special_AllPages.xhtml#Talk_INES_Mapper_048)

Mapper 48 may need a submapper assigned to it to be able to display the graphics in The Jetsons correctly as opposed to The Flintstones and other games using Mapper 48. My support is here : [http://forums.nesdev.org/viewtopic.php?f=3&t=18277](http://forums.nesdev.org/viewtopic.php?f=3&t=18277)

## CHR Reg 0 (2k @ $0000) and CHR Reg 1 (2k @ $0800)

This mapper is very similar to MMC3 but lower bit is not ignored in those registers like it's done on MMC3.
