# NES 2.0 Mapper 374

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/NES_2.0_Mapper_374) | View [other pages](Special_AllPages.xhtml#NES_2_0_Mapper_374)

**NES 2.0 Mapper 374** denotes reset-based multicarts around the [MMC1](MMC1.xhtml "MMC1") ASIC. When an interruption of the M2 signal is detected, the higher address lines are automatically incremented. So far, only 128 KiB PRG+128 KiB CHR size variants have been found: 

  * _1995 Super HiK 4-in-1 - 新系列機器戰警组合卡 (JY-022)_
  * _1996 Super HiK 4-in-1 - 新系列超級飛狼組合卡 (JY-051)_
