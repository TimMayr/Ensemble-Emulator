# 7404

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/7404) | View [other pages](Special_AllPages.xhtml#7404)

The **7404** (variants 74LS04, 74HC04) is a 74-series logic IC with six independent inverters (NOT gates). 

## 7404 Pinout
    
    
            .--\/--.
      1A -> |01  14| -- +5V
      1Y <- |02  13| <- 6A
      2A -> |03  12| -> 6Y
      2Y <- |04  11| <- 5A
      3A -> |05  10| -> 5Y
      3Y <- |06  09| <- 4A
     Gnd -- |07  08| -> 4Y
            '------'
    

## Signal descriptions

Y is the logical inverse of A (1 if A is 0 and vice versa).
