# User:Zzo38/Elemental Solitaire

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/User%3AZzo38/Elemental_Solitaire) | View [other pages](Special_AllPages.xhtml#User_Zzo38_Elemental_Solitaire)

This is a solitaire card game invented by Adrian Morgan. I am making the Famicom implementation. 

## Overview

  * Project owner: [zzo38](User_Zzo38.xhtml "User:Zzo38")
  * Developer: zzo38
  * Artist: zzo38
  * Music: zzo38
  * Music engine: Simple monophonic [just intonation](https://en.wikipedia.org/wiki/Just_intonation "wikipedia:Just intonation") engine
  * Compiler: Unofficial MagicKit
  * Other tools: Famitile
  * Mapper: NROM
  * Mirroring: Doesn't care
  * Started on: August 2012
  * Status: In progress
  * Expected date: Unknown
  * Will make cart: Probably not.
  * PRG-ROM: 8 KiB
  * CHR-ROM: Same as PRG-ROM (it is a duplicate ROM)
  * PRG-RAM: 0
  * CHR-RAM: 0
  * Players: 1
  * Controllers: Standard
  * License: Public domain (including source-codes)
  * Tested on official hardware: No (anyone who can do so, please do)
  * Action 53 compatibility: Probably
