# NES 2.0 Mapper 367

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/NES_2.0_Mapper_367) | View [other pages](Special_AllPages.xhtml#NES_2_0_Mapper_367)

NES 2.0 Mapper 367 denotes a _7-in-1_ multicart using an unknown PCB ID that constitutes a variant of [INES Mapper 205](INES_Mapper_205.xhtml "INES Mapper 205"). It latches the _address_ bits A0 and A1 instead of the _data_ bits D0 and D1 when address range $6000-$7FFF is written to, and is otherwise identical to mapper 205.
