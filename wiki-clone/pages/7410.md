# 7410

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/7410) | View [other pages](Special_AllPages.xhtml#7410)

The **7410** (variants 74LS10, 74HC10) is a 74-series logic IC with three independent 3-Input NAND gates. 

## 7410 Pinout
    
    
            .--\/--.
      1A -> |01  14| -- +5V
      1B -> |02  13| <- 1C
      2A -> |03  12| -> 1Y
      2B -> |04  11| <- 3C
      2C -> |05  10| <- 3B
      2Y <- |06  09| <- 3A
     Gnd -- |07  08| -> 3Y
            `------'
    

## Signal descriptions

Y is low if all of A, B and C are high, and is high if any of A, B or C is low.
