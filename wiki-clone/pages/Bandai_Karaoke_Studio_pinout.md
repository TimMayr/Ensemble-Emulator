# Bandai Karaoke Studio pinout

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Bandai_Karaoke_Studio_pinout) | View [other pages](Special_AllPages.xhtml#Bandai_Karaoke_Studio_pinout)

Bandai Karaoke Studio: 30-pin card edge ([iNES Mapper 188](INES_Mapper_188.xhtml "INES Mapper 188")) 
    
    
        Gnd -- | 01  30 | -> CPU A13
    CPU A12 <- | 02  29 | -> PRG A14
    CPU A11 <- | 03  28 | -> PRG A15
    CPU A10 <- | 04  27 | -> PRG A16
     CPU A9 <- | 05  26 | -> PRG A17
     CPU A8 <- | 06  25 | -> /ExternalROMRead
     CPU A7 <- | 07  24 | <> D7
     CPU A6 <- | 08  23 | <> D6
     CPU A5 <- | 09  22 | <> D5
     CPU A4 <- | 10  21 | <> D4
     CPU A3 <- | 11  20 | <> D3
     CPU A2 <- | 12  19 | <> D2
     CPU A1 <- | 13  18 | <> D1
     CPU A0 <- | 14  17 | <> D0
        Vcc -- | 15  16 | -- Gnd
