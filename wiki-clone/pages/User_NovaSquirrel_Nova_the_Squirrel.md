# User:NovaSquirrel/Nova the Squirrel

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/User%3ANovaSquirrel/Nova_the_Squirrel) | View [other pages](Special_AllPages.xhtml#User_NovaSquirrel_Nova_the_Squirrel)

**Nova the Squirrel** is a 2018 platformer game for the NES that involves puzzle mechanics and [Kirby](https://en.wikipedia.org/wiki/Kirby%27s_Adventure "wikipedia:Kirby's Adventure")-like ability copying. 

  * Mapper: [UNROM](UxROM.xhtml "UxROM") (with WRAM), [MMC1](MMC1.xhtml "MMC1"), [MMC3](MMC3.xhtml "MMC3"), or [Action 53](Action_53_mapper.xhtml "Action 53 mapper").
  * 256 KB PRG ROM
  * 8 KB PRG RAM
  * 8 KB CHR RAM
  * Libraries used: [Pently](https://www.nesdev.org/w/index.php?title=Pently&action=edit&redlink=1 "Pently \(page does not exist\)"); [PB53](Tile_compression.xhtml "PB53"); Damian Yerrick's VWF engine, voice compression, and trigonometry routines, Famitone2
