# User:Cartlemmy/Fire Below/Stars Above

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/User%3ACartlemmy/Fire_Below/Stars_Above) | View [other pages](Special_AllPages.xhtml#User_Cartlemmy_Fire_Below_Stars_Above)

Fire Below/Stars Above (Aka Hell Below/Stars Above) is the working title of a platformer puzzle adventure game for the NES. 

## Overview

  * Developer: [Cartlemmy](User_Cartlemmy.xhtml "User:Cartlemmy") (Yibbleware)
  * Assembler: ASM6
  * Other tools: [YY-CHR](YY_CHR.xhtml "YY-CHR"), custom asm6 pre-parser, custom music editor, custom map editor
  * Started on: 2010/11
  * Current status: Work in progress
  * Expected release date: 2012
  * Will make cart: Yes
