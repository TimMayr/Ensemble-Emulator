# INES Mapper 169

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/INES_Mapper_169) | View [other pages](Special_AllPages.xhtml#INES_Mapper_169)

This mapper is used in Yuxing's ROMs. PRG ROM=1024KB,CHR ROM=0KB. see [yxleimeng.net](http://www.yxleimeng.net)(chinese) 

For some reason, GoodNES used this mapper for the singleton multicart [Contra 168-in-1](http://community.fandom.com/wiki/c:bootleggames:Contra_168_in_1 "wikia:c:bootleggames:Contra 168 in 1"), but that seems to be identical to [iNES Mapper 015](INES_Mapper_015.xhtml "INES Mapper 015").
