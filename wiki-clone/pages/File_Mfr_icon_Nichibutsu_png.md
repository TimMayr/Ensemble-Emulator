Owl icon for [mappers](Mapper.xhtml "Mapper") associated with Nihon Bussan ("Nichibutsu"). 

Like Capcom, Nichibutsu used boards manufactured by Nintendo. However, one variant of UNROM was used only for one of its games (and for a homebrew multicart).
