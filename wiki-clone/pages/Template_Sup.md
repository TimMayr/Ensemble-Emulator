# Template:Sup

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Template%3ASup) | View [other pages](Special_AllPages.xhtml#Template_Sup)

[![](../wiki-images/Template-info.svg)](File_Template_info_svg.xhtml) Template documentation

**Sup** erscripts text, i.e. 

    `{{sup|Text to be superscripted}}` produces: Text to be superscripted

This template is listed at [Wikipedia:Template:Subst](https://en.wikipedia.org/wiki/Template:Subst "wikipedia:Template:Subst") as a good candidate for substituting (i.e. {{subst:sup}} ). On a page that uses the template many times, however, this is likely to clutter and lengthen the page significantly. 

Text can also be superscripted simply by using the <sup> and </sup> tags, so that: 

    `<sup>Text to be superscripted</sup>` also produces: Text to be superscripted
