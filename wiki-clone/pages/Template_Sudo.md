A user has requested a change to the protected page Template:Sudo, as described below. An administrator will [find requests like this](https://www.nesdev.org/w/index.php?title=Category:Nesdev_wiki_edit_requests&action=edit&redlink=1 "Category:Nesdev wiki edit requests \(page does not exist\)") and handle them shortly. 

[![](../wiki-images/Template-info.svg)](File_Template_info_svg.xhtml) Template documentation

If you have an idea for a fairly uncontroversial change to a given page that you lack privileges to edit, describe the edit on the corresponding talk page and add `{{sudo}}` to get an administrator's attention. For example, you could propose a change to the [Main Page](Main_Page.xhtml "Main Page") by adding a section to [Talk:Main Page](https://www.nesdev.org/w/index.php?title=Talk:Main_Page&action=edit&redlink=1 "Talk:Main Page \(page does not exist\)") and putting `{{sudo}}` at the top of your comment.
