A testcard picture intentionally resembling Philips PM5544, but rendered with NES colors. 

Rendered in 256x240. 

This image is part of a set illustrating the differences between palette-and-pixel-based and scanline-and-ntsc-signal-based rendering in NES emulation. This image shows the palette-and-pixel-based rendering.
