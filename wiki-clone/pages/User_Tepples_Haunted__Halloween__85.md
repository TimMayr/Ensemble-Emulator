From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/User%3ATepples/Haunted%3A_Halloween_%2785) | View [other pages](Special_AllPages.xhtml#User_Tepples_Haunted__Halloween__85)

_**Haunted: Halloween '85**_ is an unlicensed NES game by Retrotainment Games released on cartridge in October 2015. 

## Tech specs

  * Memory size: 512 KiB PRG ROM, 8 KiB CHR RAM
  * Mapper: [Oversize BNROM](INES_Mapper_034.xhtml "INES Mapper 034")
  * Background compression: [PB8](Tile_compression.xhtml#PB53 "Tile compression") plus Markov encoding
  * Dynamic sprite loading for player and up to five enemies
