## Gameplay

_Ramses Game_ is based on a _Dragon Quest_ 5 (DS) minigame called Slime Smack, which is basically a touch-pen Whack-a-mole game with slimes. There is a game board with several holes. Monsters come out of the holes and you have to whack them in a specific order before the timer runs out. 

The NES doesn't have a touch-pen, so each hole is assigned a button combination (up+A, down+A, left+A, right+A, etc). Pressing a combination whacks the corresponding hole.
