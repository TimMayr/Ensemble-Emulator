An explosion from [Thwaite](User_Tepples_Thwaite.xhtml "User:Tepples/Thwaite"), for the [placeholder graphics](Placeholder_graphics.xhtml "Placeholder graphics") collection. 

Here's how I made this explosion in GIMP: 

  1. create a 72x48 pixel black image
  2. fill circles with radial blends; diameters are 8, 16, 20, 24 pixels
  3. copy the 24-pixel explosion twice more, using Curves to darken the high end
  4. add noise
  5. blow the highlight in Levels
  6. posterize to 4 levels
  7. change palette
