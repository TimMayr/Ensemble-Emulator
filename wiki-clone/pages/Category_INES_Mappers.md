# Category:INES Mappers

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Category%3AINES_Mappers) | View [other pages](Special_AllPages.xhtml#Category_INES_Mappers)

The [iNES](INES.xhtml "INES") file format uses 8-bit (0-255) [mapper](Category_Mappers.xhtml "Category:Mappers") numbers to describe any special hardware present on the cartridge. 

As of 13-nov-2011, Disch's mapper docs have been incorporated into this wiki. Assimilating them fully is an ongoing process. Any information in those docs, which may be found at [romhacking.net](http://www.romhacking.net/docs/362/), is older than what you will find here. Disch's documentation uses a formalized syntax which has been archived at [INES_Mapper_DischDocs](INES_Mapper_DischDocs.xhtml "INES Mapper DischDocs"). Additional global mapper organizational clues can be found at [Kevtris's mapper grid](http://kevtris.org/Projects/console/mappers/index.html)
