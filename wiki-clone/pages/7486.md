# 7486

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/7486) | View [other pages](Special_AllPages.xhtml#7486)

The 7486 (variants 74LS86, 74HC86) is a 74-series logic IC with four independent 2-input XOR gates. 

## 7486 Pinout
    
    
            .--\/--.
      1A -> |01  14| -- +5V
      1B -> |02  13| <- 4A
      1Y <- |03  12| <- 4B
      2A -> |04  11| -> 4Y
      2B -> |05  10| <- 3A
      2Y <- |06  09| <- 3B
     Gnd -- |07  08| -> 3Y
            `------'
    

## Signal descriptions

Y is low if A and B are the same (both low or both high). Y is high if A and B are different (A high and B low or vice versa).
