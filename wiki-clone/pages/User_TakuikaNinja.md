## NESdev Projects

### FDS

It has become pretty apparent that I like the [FDS](Family_Computer_Disk_System.xhtml "FDS") a bit more than others... 

  * [FDS-BIOS-Dumper](https://github.com/TakuikaNinja/FDS-BIOS-Dumper) \- Disk program to dump the [FDS BIOS](FDS_BIOS.xhtml "FDS BIOS").
  * [FDS-Mirroring-Test](https://github.com/TakuikaNinja/FDS-Mirroring-Test) \- Test program for PPU nametable arrangement/mirroring switches, including the previously undocumented $4030.D3 state.
  * [asm6f-fds-example](https://github.com/TakuikaNinja/asm6f-fds-example) \- Demonstration of using [asm6f](https://github.com/freem/asm6f) for FDS development, analogous to [Rainwarrior's CA65 example](https://github.com/bbbradsmith/NES-ca65-example/tree/fds)
  * [dr-mario-fds](https://github.com/TakuikaNinja/dr-mario-fds) \- Proof-of-Concept port of _Dr. Mario_.
  * [TetrisFDS](https://github.com/TakuikaNinja/TetrisFDS) \- PoC port of _Tetris_ (Nintendo).
  * [Ikki-FDS](https://github.com/TakuikaNinja/Ikki-FDS) \- PoC port of _いっき (Ikki)_.
  * [FC-DiskBASIC](https://github.com/TakuikaNinja/FC-DiskBASIC) \- Port of _Family BASIC v2.1A_ (aka _Disk BASIC_), originally documented in hacking magazines during its commercial life.



### Namco IPL Interface

These relate to an Initial Program Loader (`IPL.MAIN`) found in FDS games developed by Namco: 

  * [IPL-MAIN](https://github.com/TakuikaNinja/IPL-MAIN) \- Ongoing research/analysis of the IPL and its [ interface hardware](Namco_IPL_Interface.xhtml "Namco IPL Interface").
  * [IPL-demo](https://github.com/TakuikaNinja/IPL-demo) \- Demo program to be loaded using the IPL.
  * [IPL-BattleCity](https://github.com/TakuikaNinja/IPL-BattleCity) \- _Battle City_ ported to the FDS via the IPL.



### FC/NES

  * [smb1-bugfix](https://github.com/TakuikaNinja/smb1-bugfix) \- _Super Mario Bros._ with bugfixes, Quality-of-Life improvements & more.
  * [dr-mario-input-fixes](https://github.com/TakuikaNinja/dr-mario-input-fixes) \- _Dr. Mario_ with its controller polling routine fixed to avoid input corruption and game crashes.
  * [TetrisNESBugFix](https://github.com/TakuikaNinja/TetrisNESBugfix) \- _Tetris_ (Nintendo) with bugfixes, including an optimised scoring routine to avoid game crashes.
  * [NamcoSerial.lua](https://gist.github.com/TakuikaNinja/cf2dab7ed4d5869c4be94a1753ffa76f) \- Mesen2 Lua script to simulate a [ serial checker interface](Namco_Serial_Checker_Interface.xhtml "Namco Serial Checker Interface") used for some cartridge games developed by Namco.
