# Template:Subsub

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Template%3ASubsub) | View [other pages](Special_AllPages.xhtml#Template_Subsub)

[![](../wiki-images/Template-info.svg)](File_Template_info_svg.xhtml) Template documentation

**Sub** scripts text two times, i.e. 

    `{{subsub|Text to be subscripted twice}}` produces: Text to be subscripted twice

This is equivalent to writing [Template:Code](https://www.nesdev.org/w/index.php?title=Template:Code&action=edit&redlink=1 "Template:Code \(page does not exist\)").
