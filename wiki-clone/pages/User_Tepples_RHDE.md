# User:Tepples/RHDE

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/User%3ATepples/RHDE) | View [other pages](Special_AllPages.xhtml#User_Tepples_RHDE)

_**RHDE: Furniture Fight**_ is a 2-player real-time strategy game for NES by Damian Yerrick that answers the question "What if _The Sims_ did _Rampart_?". It placed second in the 2014 NESdev competition.[1]

  * Cartridge: NROM, 256 kbit PRG ROM + 64 kbit CHR RAM
  * Libraries used: [Pently](https://www.nesdev.org/w/index.php?title=Pently&action=edit&redlink=1 "Pently \(page does not exist\)") 0.04; [PB53](Tile_compression.xhtml "PB53"); Damian Yerrick's VWF engine
