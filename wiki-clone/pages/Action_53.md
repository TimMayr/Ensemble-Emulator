# Action 53

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Action_53) | View [other pages](Special_AllPages.xhtml#Action_53)

**Action 53** is a series of homebrew multicart releases, using a mapper designed by Damian Yerrick for this purpose. 

For the multicart mapper, see: [Action 53 mapper](Action_53_mapper.xhtml "Action 53 mapper")

  * _STREEMERZ: Action 53 Function 16 Volume One_ ([discussion](http://forums.nesdev.org/viewtopic.php?f=22&t=9393) | [download](http://forums.nesdev.org/viewtopic.php?p=109907#p109907))
  * _Double Action 53: Volume 2_ ([discussion](http://forums.nesdev.org/viewtopic.php?f=5&t=11220) | [download](http://forums.nesdev.org/viewtopic.php?p=136784#p136784))
  * _Action 53 Vol. 3: Revenge of the Twins_ ([discussion](http://forums.nesdev.org/viewtopic.php?t=15570) | [download](http://forums.nesdev.org/viewtopic.php?p=219411#p219411))
  * _Action 53 Vol. 4: Actually 54_ ([download](https://action53.itch.io/vol4))
  * _NESDev '19 Compo Cart_ ([download](https://action53.itch.io/compocart19))



This [multi-discrete mapper](Action_53_mapper.xhtml "Action 53 mapper") (iNES #28) can simulate the behavior of popular discrete boards. In the first volume, the menu and all games except _STREEMERZ_ run in this mapper's [BNROM](INES_Mapper_034.xhtml "BNROM") compatibility mode. The second and third volumes still use BNROM mode for the menu and most games but uses [UNROM](UxROM.xhtml "UxROM") mode for several games and NROM-128 mode for 16K games so that both games in a bank can have their own NMI vectors. The third volume uses [CNROM](CNROM.xhtml "CNROM") mode (which requires 32K RAM) for one game (_Sinking Feeling_)
