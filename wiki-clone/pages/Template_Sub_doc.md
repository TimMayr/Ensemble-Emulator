**Sub** scripts text, i.e. 

    `{{Sub|Text to be subscripted}}` produces:   
Text to be subscripted

This template may be listed at [Wikipedia:Template:Subst](https://en.wikipedia.org/wiki/Template:Subst "wikipedia:Template:Subst") as a good candidate for substituting (i.e. {{subst:sub}} ). On a page that uses the template many times, however, this is likely to clutter and lengthen the page significantly. 

Text can also be subscripted simply by using the <sub> and </sub> tags, so that: 

    `<sub>Text to be subscripted</sub>` also produces: Text to be subscripted
