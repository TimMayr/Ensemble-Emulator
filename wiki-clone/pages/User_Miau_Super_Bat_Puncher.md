# User:Miau/Super Bat Puncher

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/User%3AMiau/Super_Bat_Puncher) | View [other pages](Special_AllPages.xhtml#User_Miau_Super_Bat_Puncher)

Super Bat Puncher is a homebrew NES platformer. It is based on the same engine as [Project P](User_Miau_Project_P.xhtml "User:Miau/Project P"), but features free multi-directional scrolling. A demo version was released on June 1, 2011. 

## Overview

  * Developer: [miau](User_Miau.xhtml "User:Miau")
  * Assembler: CA65
  * Other tools: [YY-CHR](YY_CHR.xhtml "YY-CHR"), custom map editor
  * Started on: 2010/03
  * Current status: Work in progress
  * Expected date: Unknown
  * Will make cart: Hopefully!
