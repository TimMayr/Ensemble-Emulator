# Category:Multicart mappers

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Category%3AMulticart_mappers) | View [other pages](Special_AllPages.xhtml#Category_Multicart_mappers)

This category consists of mappers whose primary purpose was for [multicarts](Multicart.xhtml "Multicart"). (CNROM and most MMC1 boards don't count, despite _Donkey Kong Classics_ and _Super Mario Bros./Duck Hunt/World Class Track Meet_.) Most of these were only ever released with one set of games.
