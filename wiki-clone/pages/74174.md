# 74174

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/74174) | View [other pages](Special_AllPages.xhtml#74174)

The **74174** (common variants 74LS174, 74HC174) is a 74-series logic Hex D-Type Flip-Flop with clear. Several discrete logic [mappers](Mapper.xhtml "Mapper") use it as a latch. 

## 74174 Pinout
    
    
                  .--\/--.
         /Clear -o|01  16|-- +5V
             Q0 --|02  15|-- Q5
             D0 --|03  14|-- D5
             D1 --|04  13|-- D4
             Q1 --|05  12|-- Q4
             D2 --|06  11|-- D3
             Q2 --|07  10|-- Q3
            Gnd --|08  09|<- Clock
                  `------'
    

## Signal descriptions

While /Clear is low, Q is reset to 0. 

A low-to-high transition of Clock causes Q to be set to the value of D.
