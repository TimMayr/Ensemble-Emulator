From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/User%3ATepples/Concentration_Room) | View [other pages](Special_AllPages.xhtml#User_Tepples_Concentration_Room)

**Concentration Room** is a card-matching game for NES by [Damian Yerrick](User_Tepples.xhtml "User:Tepples"). 

## Features

[![](../wiki-images/Croom_screenshot01.png)](File_Croom_screenshot01_png.xhtml)

Screenshot from Concentration Room 0.01

  * Modes: Story, Solitaire, 1P vs. CPU, 1P vs. 2P
  * Control is more responsive than I Can Remember



### Free version

  * 16+8 KiB [NROM](NROM.xhtml "NROM")
  * 36 predefined emblems on cards
  * No save
  * Source code included under GNU General Public License, version 3 or later
  * Includes an example of [shuffled code](Watermarking.xhtml#Shuffling "Watermarking")
