# 7408

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/7408) | View [other pages](Special_AllPages.xhtml#7408)

The **7408** (variants 74LS08, 74HC08) is a 74-series logic IC with four independent 2-input AND gates. 

## 7408 Pinout
    
    
            .--\/--.
      1A -> |01  14| -- +5V
      1B -> |02  13| <- 4A
      1Y <- |03  12| <- 4B
      2A -> |04  11| -> 4Y
      2B -> |05  10| <- 3A
      2Y <- |06  09| <- 3B
     Gnd -- |07  08| -> 3Y
            `------'
    

## Signal descriptions

Y is high if both A and B are high. Y is low if at least one of A and B is low.
