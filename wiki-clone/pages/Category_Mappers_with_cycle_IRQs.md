# Category:Mappers with cycle IRQs

From [NESdev Wiki](Nesdev_Wiki.xhtml) | View [current version](https://www.nesdev.org/wiki/Category%3AMappers_with_cycle_IRQs) | View [other pages](Special_AllPages.xhtml#Category_Mappers_with_cycle_IRQs)

Mappers in this category have the ability to schedule an IRQ an arbitrary and continuously-adjustable number of CPU cycles in the future. Cycle counting mappers that have a fixed interval that cannot be configured belong in the [fixed-timing subcategory](Category_Mappers_with_fixed_timing_cycle_IRQs.xhtml "Category:Mappers with fixed-timing cycle IRQs"). 

## Subcategories

This category has only the following subcategory. 

### M

  * [Mappers with fixed-timing cycle IRQs](Category_Mappers_with_fixed_timing_cycle_IRQs.xhtml "Category:Mappers with fixed-timing cycle IRQs")
